# 01_PRD.md

## 1. 概要
本プロダクトは、海外暗号資産取引所および DEX シグナルを活用し、イベント発生時のみ売買する小資金向けの半自動/自動取引システムである。
主戦略は、新しめのアルトコインを対象とした先物-現物乖離の extreme 逆張りであり、Funding Rate・出来高・Open Interest・上場経過日数・DEX 側話題化を補助特徴量として用いる。

## 2. 背景
- BTC や大型銘柄の単純逆張りは、ボラティリティ低下や手数料負けの影響を受けやすい。
- 一方、新規上場・新規 perpetual 上場・話題化したアルトコインでは、短期間の価格歪みが大きくなる傾向がある。
- 小資金では常時売買より「条件成立時のみトレード」の方が実効コストを抑えやすい。

## 3. 目的
- MVP として MEXC / Bitget で dry-run と paper mode を完成させる。
- basis extreme 戦略を、自動スキャン → ランキング → 発注 → ログ保存まで一貫して動かす。
- 将来的に Bybit / OKX / Gate / CoinEx / Hyperliquid を adapter 追加だけで増設できる構造にする。

## 4. 非目標
- 初期フェーズでの DEX 実売買
- HFT / マイクロ秒単位の競争
- 複数戦略の同時最適化
- フル自律ポートフォリオ運用
- 法域や税務処理の自動最適化

## 5. 主要ユースケース
1. 指定した取引所群から現物・先物価格、Funding、板、出来高を取得する。
2. 先物-現物乖離を rolling z-score 化する。
3. 条件を満たした銘柄のみ候補として抽出する。
4. フィルタ後に最上位 1〜2 銘柄だけ注文する。
5. post-only を優先して分割発注する。
6. 利確・損切り・時間切れ・API 異常時停止を自動実行する。

## 6. 成功条件
- dry-run で 7 日以上連続稼働できる。
- adapter ごとの reconnect / resubscribe が自動復旧する。
- signal, order, fill, pnl が一貫したイベントログとして保存される。
- 設定で取引所の ON/OFF、銘柄制限、手数料上書きが可能。
